The text expand component truncates long blocks of text. In place of the truncated text, it displays an ellipsis and a link to expand the text. When users click the link to expand the text, the component expands the text inline unless the text exceeds the limits on text characters or newline characters. If the text exceeds those limits, the link opens a modal view. If the text length is less than the specified threshold, then the component does not truncate the text. However, if the text includes newlines, the text is truncated regardless of its total length. Newlines are always removed from the truncated text.

To avoid truncating text in the middle of a word, the directive looks for a space in the 10 characters before the last character.
